Commonly used html tags (HTML-теги, що часто використовуються)

Tag name (Назва тега) Description Опис
< !– – > Defines comments in an HTML document. Визначає коментарі в документі HTML.
<!DOCTYPE> Specifies the version of HTML. Визначає версію HTML.
<a> Known as the anchor tag. Creates a hyperlink or link. Відомий як тег прив’язки. Створює гіперпосилання або посилання.
<article> It defines the self-contained article or content. Він визначає самостійну статтю або вміст.
<audio> Embeds sound content in HTML documents. Вбудовує звуковий вміст у документи HTML.
<b> Makes a text bold. Робить текст жирним.
<body> Defines the body or the main content section of an HTML document. Визначає тіло або основний розділ вмісту HTML-документа.
<br> Inserts a single line break. Вставляє один розрив рядка.
<button> Represents a clickable button. Являє собою кнопку, яку можна натиснути.
<caption> Defines a caption for the table. Визначає підпис до таблиці.
<center> Aligns the content in the center. Вирівнює вміст по центру.
<code> Displays a part of programming code in an HTML document. Відображає частину програмного коду в документі HTML.
<div> It defines a division or section within an HTML document. Він визначає розділ або розділ у HTML-документі.
<em> Emphasizes the content within this element. Підкреслює вміст цього елемента.
<h1 to h6> Defines headings in an HTML document from level 1 to level 6. Визначає заголовки в документі HTML від рівня 1 до рівня 6.
<head> It defines the head of the HTML document that contains all the necessary information for the browser. Він визначає заголовок HTML-документа, який містить всю необхідну інформацію для браузера.
<html> Represents the root of an HTML document. Являє корінь документа HTML.
<i> Represents a text in some different voice or style. Представляє текст іншим голосом або стилем.
<img> Inserts an image within an HTML document. Вставляє зображення в HTML-документ.
<input> Defines an input field for the HTML form. Визначає поле введення для форми HTML.
<label> Defines text label for the input field of form. Визначає текстову мітку для поля введення форми.
<li> Represents items of a list. Представляє елементи списку.
<link> Represents a relationship between a current document and an external document or resource. Представляє зв’язок між поточним документом і зовнішнім документом або ресурсом.
<main> Represents the most important or the main content of an HTML document. Представляє найважливіший або основний вміст HTML-документа.
<ol> Defines an ordered list of items. Визначає впорядкований список елементів.
<p> Represents a paragraph in an HTML document. Представляє абзац у документі HTML.
<script> Declares the JavaScript within an HTML document. Оголошує JavaScript в документі HTML.
<style> Contains style information for an HTML document. Містить інформацію про стилі для HTML-документа.
<table> Creates a table in an HTML document. Створює таблицю в документі HTML.
<tbody> Displays the body of an HTML table. Відображає тіло таблиці HTML.
<td> Defines cells of an HTML table which contains table data. Визначає клітинки таблиці HTML, яка містить дані таблиці.
<th> Defines the header of an HTML table. Визначає заголовок таблиці HTML.
<thead> Groups the header text of an HTML table. Групує текст заголовка таблиці HTML.
<title> It Defines the title of an HTML document. Він визначає назву документа HTML.
<tr> Defines rows in an HTML document. Визначає рядки в документі HTML.
<u> Renders text with an underline. Відображає текст із підкресленням.
<ul> Defines an unordered list of items. Визначає невпорядкований список елементів.
<video> Embeds a video content in an HTML document. Вбудовує відеовміст у HTML-документ.
<wbr> Defines a position within text where a break line may be possible. Визначає позицію в тексті, де може бути можливий розрив.
